# 图标文件说明

本目录需要以下图标文件：

## 必需的图标文件：

1. **system_logo.png** (64x64px)
   - 系统Logo，用于侧边栏和认证页面
   - 建议使用蓝色调，符合医疗科技感
   - 可以是一个简单的眼睛图标或者"灵瞳"字样

2. **user_avatar.png** (64x64px)
   - 用户头像占位符
   - 建议使用通用的人像轮廓图标
   - 背景透明或白色

## 临时解决方案：

如果暂时没有合适的图标，可以：
1. 使用在线图标生成器创建简单的图标
2. 从免费图标库下载合适的图标（如：Heroicons, Lucide, Phosphor等）
3. 使用文字或Emoji作为临时替代

## 图标要求：
- 格式：PNG（支持透明背景）
- 尺寸：建议64x64px或更高
- 风格：简洁、现代、医疗科技感
- 颜色：主要使用蓝色系(#4A90E2)和白色
